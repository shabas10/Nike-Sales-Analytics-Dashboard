📊 Nike Sales Analytics Dashboard – Power BI
📌 Project Overview

This project presents an interactive Sales Analytics Dashboard built using Power BI to analyze Nike’s sales performance. It provides insights into revenue, profit, units sold, and regional trends to support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products and regions
Track profit and discount trends
Visualize customer and sales channel behavior
📊 Key Features
📈 KPI Cards for Revenue, Profit, Units Sold, and Growth
🌍 Region-wise sales analysis using geo-maps
📊 Product category and performance insights
🛒 Channel analysis (Online vs Retail)
📉 Monthly and yearly sales trends
🎛️ Interactive filters and slicers
🛠️ Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
💡 Insights Gained
Identified high-revenue regions and top-selling products
Analyzed impact of discounts on profit
Discovered sales trends across different channels
Enabled better business decisions through visualization
🚀 How to Use
Open the .pbix file in Power BI Desktop
Explore dashboards using filters and slicers
Analyze trends and insights interactively
📌 Project Outcome

This dashboard demonstrates strong data visualization, business intelligence, and analytical skills, making it useful for real-world sales performance analysis.
